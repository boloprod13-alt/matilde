# Matilde Salimei — Portfolio

Fashion design portfolio website for the **Silver Springs II** collection (Moda Milano Graduate 2026) and **Questa Casa Ha I Capelli** thesis project.

## How to deploy on GitHub Pages

1. Create a new repository on GitHub
2. Upload all files from this folder
3. Go to **Settings > Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose `main` branch and `/ (root)` folder
6. Click **Save**
7. Your site will be live at `https://your-username.github.io/repository-name/`
